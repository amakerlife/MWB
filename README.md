## 0x02 初始单词表格式

格式如下：

第一行一个大于等于 $4$ 的整数，表示单词个数
第 $2$ 到第 $(3 \times n+1)$ 行，每三行格式如下：

```tex
英文单词
词性
中文意思
```

样例：

```tex
6
apple
n.
苹果
banana
n.
香蕉
run
v.
跑
walk
v.
走
jump
v.
跳
club
n.
社团
```
